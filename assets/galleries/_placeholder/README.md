# `assets/galleries/_placeholder/`

The theme reserves this path for a placeholder gallery, but ships no
binary assets here. Group authors drop their own JPGs / PNGs / WebP
files into `assets/galleries/<slug>/` in their site repo - see
SPEC-04 and `archetypes/galleries.md`.

This README is here so the directory survives `git add`. Delete it in
your site repo if you don't need a `_placeholder` slug.
