Theme palettes

Every palette below uses only colours from the official Scouts brand guidelines. Pick the one that fits your Group, then set params.palette in your hugo.toml to the key shown on each card.

To preview a palette in context, change the value in exampleSite/hugo.toml and reload — every section on the home page will update.
